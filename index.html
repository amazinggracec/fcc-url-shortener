API Basejump: URL Shortener

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response
  • When I visit that shortened URL, it will redirect me to my original link.

Example create usage:

https://fcc-amazing-url-shortener.herokuapp.com/new/https://www.google.com

https://fcc-amazing-url-shortener.herokuapp.com/new/http://freecodecamp.com/news

If you want to pass a site that doesn't exist (or an invalid url) for some reason you can do:

https://fcc-amazing-url-shortener.herokuapp.com/new/pZZZ

Example creation output:

{"original_url":"http://freecodecamp.com/news","short_url":"http://fcc-amazing-url-shortener.herokuapp.com/3"}

Usage

http://fcc-amazing-url-shortener.herokuapp.com/3

Will redirect to:

http://freecodecamp.com/news
